# PS Portfolio

Personal portfolio site for Panashe Sanyanga, Computer Science student at Manchester Metropolitan University (graduating 2027).

**Live site:** [code-by-panashe-sanyanga.github.io/PS-PORTFOLIO](https://code-by-panashe-sanyanga.github.io/PS-PORTFOLIO/)

## Pages

- `index.html`: home, skills, featured projects, timeline
- `projects.html`: full project list
- `about.html`: about me and short project notes
- `contact.html`: contact form
- `cv.html`: printable CV

Built with HTML, CSS, and JavaScript. Hosted on GitHub Pages.

## Run locally

```bash
git clone https://github.com/code-by-panashe-sanyanga/PS-PORTFOLIO.git
cd PS-PORTFOLIO
python -m http.server 8000
```

Then open `http://localhost:8000`.
